# FNF: GMCell Engine
GMCell Engine is the unofficial Friday Night Funkin based engine that was coded to work for PS3 on Gamemaker.
The game uses AI-Generated script (charts and note system only)!!

# BUILDING
Download the GameMaker 1.4.9999 and compile it.

For PS3 Platforms use GayMaker-PS3.

> [!NOTE]
> As for Charts it currently supports Original v0.2.7 json charts.
> Please don't ask me of how to get your chart supported. It is a such a complicated issue..
